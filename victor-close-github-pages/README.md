# Victor Close Countdown

A free GitHub Pages countdown app for Uzair and Nida moving into Victor Close.

## Publish on GitHub Pages

1. Create a new GitHub repository called `victor-close-countdown`.
2. Upload all files from this folder to the repository.
3. Go to **Settings** → **Pages**.
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ root**
5. Tap **Save**.

Your free link will look like:

`https://YOUR-GITHUB-USERNAME.github.io/victor-close-countdown/`

Open the link on iPhone, tap **Share**, then **Add to Home Screen**.
